## ChaoRec（超级推荐）

ChaoRec 是基于 Python 和 PyTorch 开发的，用于在统一、全面和高效的框架内复制和开发推荐算法，以达到研究目的。

主要包括一般推荐和多模态推荐。

目前的一般推荐模型有：

- BPR
- DGCF
- NGCF
- LightGCN
- MacridVAE
- MultVAE
- SGL
- NCL
- LightGCL
- HCCF
- DCCF
- AdaGCL
- VGCL

目前的多模态推荐模型有：

- VBPR
- MMGCN
- GRCN
- MGAT
- LATTICE
- MICRO
- FREEDOM
- DualGNN
- DRAGON
- BM3
- SLMRec
- MGCL
- MGCN
- POWERec
- MMGCL
- MVGAE
- MMSSL

现有模型大部分按照原作者代码进行改写，如果发现有错误欢迎指正！
